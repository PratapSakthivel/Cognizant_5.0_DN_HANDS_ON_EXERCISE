# Module 1 - Design Patterns

This module covers mandatory hands-on exercises for Design Patterns.

## Exercises

### 1. Singleton Pattern
- **Location:** `SingletonPatternExample/`
- **Files:** `Logger.java`, `TestLogger.java`
- Demonstrates a thread-safe Singleton Logger class.

### 2. Factory Method Pattern
- **Location:** `FactoryMethodPatternExample/`
- **Files:** `Document.java`, `DocumentFactory.java`, `WordDocument.java`, `WordDocumentFactory.java`, `PdfDocument.java`, `PdfDocumentFactory.java`, `ExcelDocument.java`, `ExcelDocumentFactory.java`, `TestFactory.java`
- Demonstrates the Factory Method pattern for creating different document types.
